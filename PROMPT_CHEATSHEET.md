# DataBuddy AI - Quick Prompt Cheat Sheet

## 🚀 Copy & Paste Ready Prompts

---

## 📝 CREATE TABLE

```
Create employees table with name, email, and salary
Create products table with name, price, and stock
Create customers table with name, email, and phone
Create orders table with customer name, product, and quantity
Make a table for inventory with product name, quantity, and location
```

---

## ➕ ADD COLUMNS

```
Add email column to employees table
Add phone number and address columns
Add new columns: department, manager, and team size
Add salary column to employees
Add category column to products
```

---

## 👀 VIEW DATA

```
Show me all employees
Get all products
List all customers
Display all orders
Show me employees where salary is greater than 50000
Find products with price less than 100
Get employees where department is Sales
Show me customers where city is New York
```

---

## ➕ ADD DATA

```
Add employee: name John, email john@company.com, salary 50000

Add three employees: John 50000, Bob 60000, Alice 55000

Insert product: name Laptop, price 999, stock 50
```

## ✏️ UPDATE DATA

```
Change John's salary to 60000
Update Laptop price to 899
Modify Sarah's email to newsarah@email.com
Change all product prices to 50
Set John's department to Sales
Update stock to 25 where product is Laptop
```

## 🗑️ REMOVE DATA

```
Remove all data from employees table
Delete everything from products
Clear all data

Delete Laptop from products

Delete products where stock is 0
```

## 🎯 COMPLETE WORKFLOW EXAMPLE

**Important: Send each command separately, one at a time!**

```
Command 1: Create employees table with name, email, and salary
Command 2: Add employees: John john@company.com 50000, Bob bob@company.com 60000
Command 3: Show me all employees
Command 4: Add department column to employees
Command 5: Change John's department to Sales
Command 6: Show me employees where department is Sales
Command 7: Change John's salary to 60000
Command 8: Remove Bob from employees
```

**💡 Remember**: Send one prompt, wait for it to complete, then send the next one!
